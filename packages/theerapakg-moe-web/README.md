# theerapakg.moe

## Setup

install the dependencies

```bash
pnpm install
```

## Development

start the development servers

```bash
pnpm dev:db
pnpm dev:site
pnpm dev:app
```

or to start all development servers

```bash
pnpm dev
```

## Production

build the application for production

```bash
pnpm build:site
pnpm build:app
```
